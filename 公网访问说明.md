# 🌐 让任何人访问你的网站

## 方法一：使用 localtunnel（推荐，最简单）

### 步骤：

1. **启动开发服务器**（在一个终端窗口）：
   ```bash
   npm run dev
   ```

2. **创建公网链接**（在另一个终端窗口）：
   ```bash
   lt --port 5173
   ```

3. **复制生成的链接**，类似这样：
   ```
   your url is: https://xxxxx.loca.lt
   ```

4. **将这个链接分享给任何人**，他们就可以访问你的网站了！

---

## 方法二：使用 ngrok（更稳定）

### 安装 ngrok：
访问 https://ngrok.com/download 下载并安装

### 使用：
1. 启动开发服务器：`npm run dev`
2. 在另一个终端运行：`ngrok http 5173`
3. 复制生成的 https 链接分享给别人

---

## 方法三：部署到 Vercel（永久免费，最稳定）

### 步骤：

1. **安装 Vercel CLI**：
   ```bash
   npm install -g vercel
   ```

2. **登录 Vercel**：
   ```bash
   vercel login
   ```

3. **部署**：
   ```bash
   vercel
   ```
   按照提示操作，完成后会得到一个永久的网址，类似：
   `https://your-project.vercel.app`

4. **以后更新**：
   ```bash
   vercel --prod
   ```

---

## 📱 手机访问注意事项

- 确保手机和电脑连接的是**同一个 Wi-Fi**（局域网访问）
- 或者使用上面的方法创建**公网链接**，这样任何网络都能访问

---

## ⚠️ 重要提示

- **localtunnel** 和 **ngrok** 是临时链接，关闭后链接会失效
- **Vercel** 是永久免费的，适合长期使用
- 如果使用 localtunnel，链接可能会变化，每次重启需要重新分享
